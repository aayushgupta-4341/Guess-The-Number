## ✨ Features

* 🎯 Random number generation  
* 🔢 Guess between dynamic ranges (based on difficulty level)  
* 📊 Attempts counter  
* 🏆 Best score tracking (using localStorage)  
* 🎚️ Difficulty Levels:
  * Easy (1–50) → Strong hints  
  * Medium (1–100) → Balanced hints  
  * Hard (1–200) → Minimal hints  
* 💬 Smart hints (Too High / Too Low / Close)  
* 🔄 Restart game functionality  
* 🌗 Dark / Light mode toggle (icon-based)  
* 📱 Fully responsive and clean UI
